# Data Sources

This dataset includes multiple types of data:

## 1. Real Measurements
Collected manually from Pixel 6a usage.

## 2. Synthetic Data
Generated using thermal resistance models:

T_battery ≈ 0.88 * T_ambient + 9.5 (normal use)

## 3. Literature-based Data
Derived from:
- Notebookcheck thermal measurements
- User reports (Reddit, forums)
- Smartphone thermal modeling studies

All synthetic data is clearly labeled.

## Synthetic Data Notice

※このデータは実測ではなく、公開論文・ユーザー報告を基にした統計シミュレーションです。

This dataset includes synthetic data generated using thermal models and literature-based parameters. It is not fully measured real-world data.

Synthetic values are used for:
- Filling missing conditions
- Baseline modeling
- Comparative analysis

All synthetic entries are explicitly labeled in the `source_type` column.

# Data Source Definitions

## Real Data
Directly measured from Pixel 6a usage.

## Derived Data
Calculated from partial measurements or interpolation.

## Synthetic Data
Generated using simplified thermal models:

T_battery ≈ 0.88 × T_ambient + 9.5

## Important Notice

Synthetic data is NOT ground truth and is used only for:
- baseline modeling
- missing condition estimation
- comparison reference
